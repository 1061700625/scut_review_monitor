# scut_blind_monitor
华南理工大学论文盲审状态监控脚本


## 使用方法
1. 安装环境

```bash
pip install -r requirements.txt

playwright install chromium
```

2. 运行脚本
```bash
python monitor.py
```
首次使用会弹出浏览器，需要扫码登陆。
